sgen

(javascript) string generator

Github project

Timestamp
Random

installation

component/component 

$ component install [--save/--save-dev] ramitos/sgen

npm

$ npm install [--save/--save-dev] sgen

example

var sgen = require('sgen');

sgen.timestamp();
sgen.random();

sgen.timestamp(new Date(2012, 1, 1).getTime());
sgen.random(12);
